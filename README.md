# Form-Prototype-Builder

The project aims to simplify form prototyping, an essential step that should be pursued prior to jumping to the code.

CURRENT FEATURES:
- Drag and Drop fields
- Edit fields
- Remove fields
- Reorder fields
- Export form as .png image
